## 🔧 How agent_plus Works

agent_plus uses a system called **Skills**. Skills are like built-in helpers that do specific jobs for you.

For example, some skills let the app understand files and answer questions about them. Others add middle steps to handle tasks more cleanly.

This system helps the app work smoothly and add more features later.

---

## 🗂 Skills Structure Explained

The skills in agent_plus are stored in folders with a clear layout:

```
skill-name/
├── SKILL.md          (Main skill description)
├── reference.md      (Optional extra info)
├── resources/        (Supporting files)
```

Each skill has a detailed description file that explains what it does and how it works.
